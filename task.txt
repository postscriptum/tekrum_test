Тестовое задание:

Создать веб приложение (виртуальная корзина) со следующим функционалом:
- на главной страничке отобразить кнопку "Создать покупку" при нажатии на которую реализовать переход на страничку, где можно создать покупку.
В качестве товара использовать любые 5 продуктов (например, хлеб, молоко, масло, мясо, сыр). В покупке может быть только один товар одного вида.
- на главной страничке вывести список покупок в формате дата + купленные товары + общая сумма покупки + плюс иконка для удаления покупки.
- при клике на покупку - переход на страничку для создания покупки, где можно отредактировать (добавить/удалить продукт) покупку

При создании приложения использовать следующие технологии:
Обязательные (5 балла):
- Django Rest Framework
- Vue.js
Желательные:
- Vuex (https://vuex.vuejs.org/en/) (1 балл)
- Iview (https://www.iviewui.com/) (1 балл)

Развернуть проект на любом бесплатном сервере (heroku, pythonanywere и тп) (3 баллов)
Выполненное задание должно быть прислано в качестве ссылки на github репозиторий с исходным кодом
